# Bush Tweaks

A small mixin mod that makes *Berries & Cherries* bushes behave like vanilla sweet-berry bushes: crouch-safe, correct slowdown, and proper damage gating.

- **Minecraft:** 1.21.1
- **Loader:** NeoForge
- **Mod ID:** `bushtweaks`
- **Dependency:** Requires the **Berries & Cherries** mod to have any effect.

Contains only original patch code; it references the target mod's classes by name via mixin.
